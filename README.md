# Gocker

Скромная, неполноценная, но крайне душевная мини-реализация Docker на Python с использованием системных вызовов Linux. Проект создан исключительно в учебных целях для изучения низкоуровневых механизмов контейнеризации.

---

## Возможности

* Изоляция процессов. Контейнер видит только себя и запущенную команду.
* Изоляция файловой системы и монтирование независимой директории `/proc`.
* Изоляция имени хоста (hostname).
* Скачивание манифестов и слоев образов напрямую через Docker Registry API V2 (с поддержкой архитектуры `amd64`).
